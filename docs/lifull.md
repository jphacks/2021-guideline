## LIFULL API 利用方法

- トークン取得用URI:
```
https://auth.homes.co.jp/token
```

- 利用可能APIサーバー:
```
https://api.homes.co.jp/
```

- ClientID:
```
      YjE0YzU2MmNiOTk2MzI4ZjkzMzcwM2ZkNjEzMGU2
```

- ClientSecret:別途メールにてお送りします。

- 無料電話番号(賃貸・流通)：なし
- 無料電話番号(分譲)：なし
- 問い合わせAPI：利用不可
